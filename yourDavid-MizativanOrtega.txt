    37. Walk me through Lambda's git flow. 
    Create your own version of the Git for Web Development Project repo by forking it. Click settings and go to Manage access, add your TL as a collaborator. Go to your repo and click on the green button Clone or download. Click the notepad which will copy the url. Go on your command line and go into the directory where you want to clone this repo by using the cd command. Then write git clone "link to repo". Cd into your cloned repo name. Create a branch: git checkout -b "firstName-lastName". Add a file to this project called yourFirstName-yourLastName.txt. You want to add and commit the changes so write git add. and then git commit -m "Your helpful commit message here". Then you need to push your changes by git push -u origin branch-name. Now you go back to your forked project repo on github and click Branch:master button on the left side. It creates a dropdown where you should see your branch name. Click your name-branch and then click the New pull request button to make a pull request on your name branch. Click on the button that says base repository and change it to your forked repo. Add your TL to the reviewers and then submit the pull request.

Stretch Questions

    1. What is the difference between an inline element and a block element?
    Inline elements do not force a new line to being in the document flow and they can be nested inside of other elements without disrupting the flow of content. A block element causes a line break to occur.

    2. What happens when an element is positioned absolutely? 
    It becomes an element whose computed position value is absolute or fixed.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    box-sizing: border-box 

    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
    block elements: div, p, li.
    inline elements: <a>, <span>.
    inline block element: img.
